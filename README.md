
Ciggycraft
=======
<img src="https://cdn.modrinth.com/data/EqIpFduf/images/897348897ae487eee577ef45d4a8385db2e3eb47.png" style="width:200px"><br>
This is a Java mod for Minecraft that adds tobacco products.

More information on Modrinth: https://modrinth.com/mod/ciggycraft

Available under MPL-2.0 as required by Tide 2, which is sourced in order to provide backport behavior for bundles that are used by PackOfCigarettes in 1.21.1.
